Descripción del modulo 

En este modulo se centralizan todos los recursos virtuales que maneja el estudiante en apoyo a las clases que ha cursado o esta cursando en base a su plan académico. 
El estudiante puede buscar y visualizar los distintos recursos literarios en base a las asignaturas de su carrera, además de ver la descripción del mismo, como ser: 
titulo, autor, tags entre otros, y la visualización del recurso.

Restricciones 
- Se deben mostrar los recursos en base a clases aprobadas o que se están cursando.
- El archivo debe mostrar las tags.
- Se puede visualizar el archivo pero no descargarse.


Requerimientos para el modulo Biblioteca Virtual 
Consulta SQL para buscar el recurso 
Lista de clases aprobadas por el estudiante
Recursos:
	- Titulo
	- Autor
	- Tipo de Archivo
	- Etiquetas
	- Categoría (Documento)
	- Url
